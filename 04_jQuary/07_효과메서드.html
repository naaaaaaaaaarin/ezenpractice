제이쿼리 효과

제이쿼리에서 화면전환, 슬라이드 다운과 업 등
다양한 효과 메서드를 제공

1. show / hide / toggle 효과

선택요소를 숨기거나 노출하고 싶을 경우에 사용
형식 : hide('시간') | show('시간') | toggle('시간')
인자값 '시간' -> 'fast',200 / 'normal','400' / 'slow',600 / 밀리세컨 입력

제이쿼리 효과

제이쿼리에서 화면전환, 슬라이드 다운과 업 등
다양한 효과 메서드를 제공

2. slideUp / slideDown / slideToggle 효과

slideUp : 선택요소를 위로 미끄러지듯 말아 올리며 숨김
slideDown : 선택요소를 아래로 미끄러지듯 말아내리면서 보임
slideToggle : 자동으로 인식하여 Up, Down이 자동으로 적용됨

3. fadeIn / fadeOut / fadeToggle 효과

fadeIn : 선택요소를 천천히 나타나게 함
fadeOut : 선택요소를 천천히 사라지게 함
fadeToggle : fadeIn + fadeOut
