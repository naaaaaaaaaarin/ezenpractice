CSS(Cascading Style Sheet)

웹표준에 의한 웹문서는 디자인과 내용이 분리되어 있습니다.

내용과 구조는 HTML을 이용해 구성하고, 디자인은 CSS를 이용해 꾸미는 것 입니다.

인라인 스타일시트 적용하기

class 스타일시트

class로 스타일 정의시 구분기호로 .을 사용

스타일 캐스케이딩 원리

같은 이름의 선택자에 같은 속성명이 정의 되어 있을 때,
속성값이 다르다면, 하단에 있는 스타일의 속성값 적용

ID 스타일시트

id로 스타일 정의시 구분기호로 #을 사용

그룹선택자

그룹선택자 구분기호는 ,를 사용하여 여러개의 요소를 선택
